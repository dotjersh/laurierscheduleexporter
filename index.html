LORIS Exporter

This is a webpage scraper that uses the source code of your student schedule LORIS page. Never trust any website that asks you to copy and past the source code to a webpage. That includes this one. Before you do the below, check the source code for yourself at github.com/dotjersh/lorisexporter to confirm the script is not saving, taking, or recording any of your data.
  1. Go to Student Detail Schedule in Loris
  2. Right click and select, View Page Source
  3. Copy page source and paste in box below
  4. Click Download
  5. Open downloaded file in desired Calendar Software
