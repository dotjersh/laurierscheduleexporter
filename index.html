Laurier Schedule Exporter

This webpage will generate a calendar .ics file that you can use in just about any modern calendar software. Never trust any website that asks you to copy and past the source code to a webpage. That includes this one. Before you follow the instructions below, check the source code for yourself at github.com/dotjersh/lorisexporter to confirm the script is not saving, taking, or recording any of your data.

  1. Go to students.wlu.ca and select the MY LAURIER dropdown
  2. Use the provided links to login to students.wlu.ca.
  3. Select the MY LAURIER dropdown again. If you don't see yoru timetable, follow the instructions provided to display this information.
  4. Go to students.wlu.ca/_api/api.php. You should see your timetable. If you do not, start back at step one.
  5. Right click and select, View Page Source
  6. Copy page source and paste in box below
  7. Click Download
  8. Open downloaded file in desired Calendar Software
